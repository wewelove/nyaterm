# 主题与外观

NyaTerm 支持对工作区外观做细粒度调整，包括 UI 主题、终端主题、字体和光标样式。

## UI 主题与终端主题

在 **设置 → 外观** 中，你可以分别配置：

- **界面主题** — 控制应用整体配色
- **终端主题** — 控制终端区域配色；也可以选择跟随 UI 主题

如果你只是想快速切换主题，也可以使用顶部菜单中的 **View → Theme**。

NyaTerm 内置了一个名为 **Nya High Contrast** 的高对比度主题，适合需要无障碍辅助、明亮环境或投屏场景下增强可读性时使用。它和其他 UI 主题一样，可以在 **设置 → 外观** 或顶部菜单 **View → Theme** 中选择。

## 背景图

在 **Settings → Appearance** 中，主窗口现在可以使用本地背景图。相关设置建议直接按界面里的真实名称理解：

- `Background Image`：选择主窗口背后的本地图片
- `Image Sizing`：控制图片以 `cover`、`contain`、`stretch` 或 `tile` 的方式铺开
- `Image Opacity`：控制背景图透过主题显示的强度
- `Background Content Opacity`：控制工作区面板和内容表面的透光程度

这个能力只作用于主窗口工作区。设置页和其他子窗口仍然保持实色背景，避免在表单、弹窗和独立窗口里牺牲可读性。

## 字体与字号

在 **设置 → 外观** 中可以调整：

- **字体族** — 支持主字体 + 多级回退字体
- **终端字体大小**
- **界面字体大小**

NyaTerm 内置了这些常用字体：

- `JetBrains Mono`
- `Noto Sans SC Variable`
- `Inter`

同时也会读取系统已安装字体，供你加入字体回退链路。

系统字体现在是异步发现的，因此首次展开字体列表时，界面可能会短暂显示 `Loading system fonts...`。这表示应用正在后台整理系统字体，不影响你先选择内置字体或已有回退链路。

## 光标

外观设置还支持这些终端细节：

- **光标样式**：Block / Underline / Bar
- **光标闪烁**

如果你经常在深浅主题之间切换，建议把终端主题和 UI 主题一起调整后再观察关键字高亮、动作链接等终端装饰效果。

## 语言切换

NyaTerm 当前提供：

- 简体中文
- English

你可以通过两种方式切换：

- **设置 → 通用 → 语言**
- 顶部菜单 **View → Language**

## 面板与工作区外观

除了配色与字体，工作区本身也支持按使用习惯调整：

- 左右侧面板宽度可拖动调整
- 标签内分屏比例可拖动调整
- 左右活动栏可通过快捷键快速显示 / 隐藏

这些布局状态会跟随应用设置一起保存，适合按你自己的终端工作流长期固定下来。

## 缩放与快捷操作

应用提供这些常见快捷键：

- **放大** — `Ctrl / Cmd + =`
- **缩小** — `Ctrl / Cmd + -`
- **重置** — `Ctrl / Cmd + 0`

这些入口适合在演示、投屏或高分屏环境下快速调整可读性。
