# 主题与外观

NyaTerm 支持对工作区外观做细粒度调整，包括 UI 主题、终端主题、字体和光标样式。

## UI 主题与终端主题

在 **设置 → 外观** 中，你可以分别配置：

- **界面主题** — 控制应用整体配色
- **终端主题** — 控制终端区域配色；也可以选择跟随 UI 主题

如果你只是想快速切换主题，也可以使用顶部菜单中的 **View → Theme**。

## 字体与字号

在 **设置 → 外观** 中可以调整：

- **字体族** — 支持主字体 + 多级回退字体
- **终端字体大小**
- **界面字体大小**

NyaTerm 内置了这些常用字体：

- `JetBrains Mono`
- `Noto Sans SC Variable`
- `Inter`

同时也会读取系统已安装字体，供你加入字体回退链路。

## 光标与连字

外观设置还支持这些终端细节：

- **光标样式**：Block / Underline / Bar
- **光标闪烁**
- **字体连字**

如果你经常在深浅主题之间切换，建议把终端主题和 UI 主题一起调整后再观察关键字高亮、动作链接等终端装饰效果。

## 语言切换

NyaTerm 当前提供：

- 简体中文
- English

你可以通过两种方式切换：

- **设置 → 通用 → 语言**
- 顶部菜单 **View → Language**

## 面板与工作区外观

除了配色与字体，工作区本身也支持按使用习惯调整：

- 左右侧面板宽度可拖动调整
- 标签内分屏比例可拖动调整
- 左右活动栏可通过快捷键快速显示 / 隐藏

这些布局状态会跟随应用设置一起保存，适合按你自己的终端工作流长期固定下来。

## 缩放与快捷操作

应用提供这些常见快捷键：

- **放大** — `Ctrl / Cmd + =`
- **缩小** — `Ctrl / Cmd + -`
- **重置** — `Ctrl / Cmd + 0`

这些入口适合在演示、投屏或高分屏环境下快速调整可读性。
